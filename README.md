# sindrenm's dotfiles

## Steps to set up

1. Install [dots].

2. Clone this repository:

   ```
   git clone https://github.com/sindrenm/dotfiles $HOME/.local/etc
   ```

3. Create configuration groups as you see fit. All directories aside from the
   `base` directory should contain a second directory in them representing a
   sub-configuration group.

4. Execute `dots groups set [list of groups here]` then `dots install`.

5. Enjoy your organized dotfiles!

[dots]: https://github.com/EvanPurkhiser/dots
